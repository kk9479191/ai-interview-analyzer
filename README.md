🤖 AI Interview Analyzer

An AI-powered web application that analyzes interview responses using emotion detection, speech recognition, and intelligent scoring.

🚀 Features

- 🎤 Speech-to-text answer input
- 😊 Real-time emotion detection using Face-api.js
- 🧠 AI-based answer analysis using TextBlob (NLP)
- 📊 Automated scoring system
- 📈 Dashboard to view previous interview results
- 💾 Data storage using MySQL (for structured result management)

🛠 Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Flask (Python)  
- AI & ML:  Face-api.js, TextBlob  
- Database: MySQL  
- Deployment: PythonAnywhere  

⚙️ How It Works

1. User enters their name and starts the interview  
2. Camera captures facial expressions in real-time  
3. Speech recognition converts spoken answers into text  
4. Emotion detection analyzes facial expressions  
5. NLP evaluates answer quality  
6. System generates a score and result (Selected/Rejected)  
7. Data is stored and displayed on the dashboard
